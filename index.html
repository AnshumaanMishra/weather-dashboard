About:

This website is a simple implementation of the WeatherAPI, a free-to-use api used to fetch weather details from the web. The website fetched the weather of the entered location and displays it.
The data can be configured to use either the Metric System, or the Imperial System. The Metic system uses SI units, whereas the Imperial System uses the FPS unit system. They can be adjusted by the toggle with °C (or °F) written on it
Further, this project also allows the user to see the forecast of the entire week or only the real-time weather. This can be configured using the toggle with D (or W) witten on it.
